第三步
代码加上蓝牙功能， 开发板要能定时扫描蓝牙设备，然后自动连接蓝牙键盘和蓝牙鼠标，当然不可能是个蓝牙键盘和蓝牙鼠标就进行连接。我的方案是当设别离得近也就是半米吧，才进行连接，可以用信号强度判断，这个是不是挺靠谱的。如果信号强度大于某个值，设备进行主动连接。然后把设备的行为 比如连接的蓝牙鼠标，鼠标移动或按键的行为通过usb模拟鼠标传给电脑，连接的蓝牙键盘，键盘的按键行为通过usb模拟键盘传给电脑。

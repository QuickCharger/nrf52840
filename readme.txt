还有就是
鼠标-主设备-从设备-电脑 这个链路的延迟会怎样
  蓝牙鼠标→nRF：约 5～10ms（蓝牙 BLE 1M，100Hz～1kHz）
  nRF 内部处理 + 坐标累加：<1ms
  nRF→USB HID→电脑：约 1～8ms（取决于 USB 轮询率：125Hz≈8ms，1000Hz≈1ms）
  系统 + 显示器：16.7ms（60Hz）/ 6.9ms（144Hz）
延迟可以再降低些吗
  总延迟
  6ms ~ 10ms
还有个问题 如果屏幕边缘的事件 是从机发送给主机， 这样的话，链路是不是变得更长了。 你有好的解决方案吗
  链路永远是：鼠标→主机→从机→电脑
